# 智教云 - 课程知识点 AI 教学平台

## 项目概述

为理工科教师提供一键生成课程知识点的互动式教学文档的平台。

**核心功能：**
- 输入课程 + 知识点 → AI 生成完整教学 HTML
- HTML 包含：概念、讲解、重难点、可交互仿真、总结、习题
- 支持局部 AI 修改
- 会员制付费

---

## 目标用户

- 理工科教师（物理、数学、电子、化学等）
- 需要快速制作教学课件
- 希望有互动仿真效果提升教学体验

---

## 核心业务流程

```
┌─────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────┐
│  注册   │ →  │ 输入课程    │ →  │ AI 生成     │ →  │ 预览    │
│ 登录    │    │ + 知识点    │    │ HTML 文档   │    │ 编辑    │
└─────────┘    └─────────────┘    └─────────────┘    └────┬────┘
                                                          │
                              ┌───────────────────────────┘
                              ↓
                    ┌─────────────────┐
                    │  局部 AI 修改   │
                    │  导出/下载 HTML │
                    └─────────────────┘
```

---

## 功能模块

### 1. 用户系统
- 邮箱/手机号注册登录
- 密码找回
- 个人信息管理
- 会员状态查询

### 2. 文档生成（核心）
**输入：**
- 课程名称（如：大学物理）
- 知识点（如：牛顿第二定律）
- 可选：难度等级、适用年级

**输出 HTML 结构：**
```html
<div class="knowledge-doc">
  <section id="concept">知识点概念</section>
  <section id="explanation">详细讲解</section>
  <section id="key-points">重难点分析</section>
  <section id="simulation">可交互仿真</section>
  <section id="summary">总结</section>
  <section id="exercises">相关题目及答案</section>
</div>
```

### 3. 仿真生成
- AI 判断知识点类型
- 生成对应仿真代码（Canvas/Three.js）
- 嵌入 HTML，支持参数调节

### 4. 局部修改
- 用户选中某 section
- 输入修改要求
- AI 仅修改该部分
- 保留其他内容不变

### 5. 文档管理
- 历史文档列表
- 查看/编辑/删除
- 导出 HTML/PDF
- 搜索/筛选

### 6. 付费系统
| 类型 | 权益 |
|------|------|
| 免费用户 | 每月 3 次生成 |
| 会员（¥68/月） | 每月 50 次生成 |
| 超额 | ¥2/次 |

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue3 + TypeScript + Vite + Tailwind CSS |
| 后端 | Python + FastAPI + SQLAlchemy |
| 数据库 | PostgreSQL |
| AI | Kimi / Claude / GPT-4（多模型策略） |
| 部署 | Docker + Docker Compose |

---

## 仿真类型映射

| 学科 | 仿真类型 | 技术方案 |
|------|---------|---------|
| 物理力学 | 运动仿真 | Matter.js / Planck.js |
| 电路电子 | 电路仿真 | CircuitJS / 自定义 Canvas |
| 数学 | 函数可视化 | Chart.js / 自定义 Canvas |
| 波动光学 | 波动模拟 | Canvas API |
| 化学 | 3D 分子结构 | Three.js |

## 测试用例

**牛顿第二定律**
- 课程：大学物理
- 知识点：牛顿第二定律 F=ma
- 期望仿真：可调节质量 m 和力 F，观察加速度 a 的变化，实时显示运动动画
